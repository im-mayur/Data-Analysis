Data Analysis

This folder contains Jupyter notebooks for learning and exploring data analysis and visualization libraries in Python. Each notebook focuses on a specific library or concept and includes practical examples.

Contents

NumPy Notebooks:
Introduction to NumPy arrays.
Array manipulations, indexing, slicing, and reshaping.
Statistical and mathematical operations with NumPy.

Pandas Notebooks:
DataFrame and Series operations.
Data cleaning, transformation, and manipulation.
Handling missing data and working with CSV/Excel files.

Matplotlib Notebooks:
Creating basic plots: line, scatter, bar, and pie charts.
Customizing plots with labels, titles, legends, and colors.
Advanced plotting techniques and subplots.

Seaborn Notebooks:
Statistical data visualization with Seaborn.
Exploring different plot types: heatmaps, pair plots, KDE plots, etc.
Customizing and styling Seaborn plots.

Usage
Open the notebooks in Jupyter Notebook, JupyterLab, or any compatible IDE.
Each notebook is self-contained, with step-by-step explanations and code examples.
Prerequisites
Ensure you have the following libraries installed in your Python environment:

NumPy
Pandas
Matplotlib
Seaborn
